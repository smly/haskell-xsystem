h1. sys35tools

http://github.com/smly/sys35tools

h2. DESCRIPTION

sys35tools provides some tools for system3.x (the engine AliceSoft had used).

h3. aldtool

<pre>
<code>
$ ./aldtool ./data/mamatoto_ga.ald ./data/mamatoto_gb.ald
</code>
</pre>

h3. vsp2ppm

<pre>
<code>
$ ./vsp2ppm ./data/CG_00001.VSP ./data/CG_00001.PPM
</code>
</pre>

h2. TEST

<pre>
<code>
$ cabal configure -ftest
</code>
</pre>

h2. REQUIREMENTS

* bytestring-mmap >= 0.2
* bytestring >= 0.9
