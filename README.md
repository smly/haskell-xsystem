WIP: haskell-xsystem
====================

Description
-----------

Sys35tools provides some tools to manipulate a data of system3.x (the engine AliceSoft had used).

Usage
-----

### aldtool

    $ ./aldtool ./data/mamatoto_ga.ald ./data/mamatoto_gb.ald

### vsp2ppm

    $ ./vsp2ppm ./data/CG_00001.VSP ./data/CG_00001.PPM

### Requirement

    * bytestring-mmap >= 0.2
    * bytestring >= 0.9
